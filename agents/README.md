# Agents

Language-specific dependency analysis agents for DepSentinel.

## Supported Languages

- **JavaScript/TypeScript** - npm, yarn, pnpm package analysis
- **Python** - pip, poetry, conda dependency analysis
- **Java** - Maven, Gradle dependency analysis
- **Go** - go.mod analysis
- **Rust** - Cargo.toml analysis
- **Docker** - Dockerfile and image dependency analysis

## Structure

Each agent provides:
- Dependency graph extraction
- Version compatibility checking
- Breaking change prediction
- Security vulnerability detection
