# DepSentinel Server

REST API server for DepSentinel.

## Endpoints

- `/api/projects` - Project management
- `/api/dependencies` - Dependency analysis
- `/api/vulnerabilities` - Security scanning
- `/api/predictions` - Breaking change predictions
- `/api/fixes` - Auto-generated fixes

## Development

```bash
npm run dev
```

Server runs on http://localhost:3000
